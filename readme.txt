- Chương trình đã có sẵn model đã được huấn luyện trước, toàn bộ các model nằm trong file models
- Để chạy chương trình cd đến thư mục NguyenTanTien, chạy lệnh: python src/use_model.py. Hoặc nếu đang ở thư mục NguyenTanTien/src thì chạy python use_model.py

  + python src/train.py
  + python src/evaluate.py
